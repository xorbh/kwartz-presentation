08
Gatekeeper

Secure Access Control
for Your Data

One dashboard to manage AI data access across your organisation. Gatekeeper sits between your people and your data — every time an AI tool requests information, Gatekeeper checks: does this person have permission?

How It Works

  • Define Privileges — create named access levels (Sales Viewer, Production Manager, Finance Team)
  • Assign to People — give each person only the privileges they need
  • Adjust Anytime — changes take effect immediately, no restarts

Key Benefits

  • Least-privilege access — people only see what their role requires
  • Centralised control — one dashboard for all data sources
  • Zero friction — works silently in the background
  • No data stored — only manages permissions, never sees your data
09
Gatekeeper

What You See in the
Dashboard

  • Apps — each data source connected to Gatekeeper. See exactly what data points each source exposes.
  • Privileges — your access bundles. See which data points are in each privilege and how many people have it.
  • Users — your team. See what privileges each person has and the full list of data they can access.
  • Change Log — when a data source updates, see a clear summary of what changed. Review and dismiss when satisfied.

Authentication via Clerk. Every data request checked against your permission rules. Gatekeeper never sees, stores, or caches your actual business data. Stale permissions automatically cleaned up.
